# Bedingungen

if i == 5: = Es wird geprüft, ob i genau den Wert 5 hat. Wenn dies stimmt, wird der eingerückte Code ausgeführt.

else: = Falls die Bedingung davor nicht stimmt, wird stattdessen dieser eingerückte Code ausgeführt.

# Schleifen

for i in range(10): = Wiederholt den eingerückten Code 10 Mal. Die Variable i zählt dabei von 0 bis 9.

while i < 10: = Wiederholt den eingerückten Code solange die Bedingung wahr ist. In diesem Beispiel solange i kleiner als 10 ist.

for i in range(10): = Die Variable i wird bei jedem Durchlauf automatisch verändert und zählt von 0 bis 9.

Beispiel:

for i in range(3):
    print(i)

# Variablen

name = "Max" = Speichert den Text "Max" in der Variable name.

alter = 12 = Speichert die Zahl 12 in der Variable alter.

name = "Max" = Erstellt die Variable name und speichert darin den Text "Max".

alter = 12 = Erstellt die Variable alter und speichert darin die Zahl 12.

i = 0 = Erstellt die Variable i und speichert darin die Zahl 0.

Variable = Eine Variable ist wie eine beschriftete Kiste, in der Werte gespeichert werden können.

Beispiele:

name = "Anna" = Die Variable name enthält den Text "Anna".

punkte = 100 = Die Variable punkte enthält die Zahl 100.

leben = 3 = Die Variable leben enthält die Zahl 3.

x = 5 = Die Variable x enthält die Zahl 5.

y = 10 = Die Variable y enthält die Zahl 10

Erklärung: erstes Zeichen muss ein Buchstabe sein!!!

# Häufig verwendete Variablennamen
